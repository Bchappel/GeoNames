Running:

This program must be run on a UNIX system.

Ensure the current working directory is the 'GeoNames' folder

Execute the following commands in order to run the program:

chmod 777 main.py
./main.py -a aus.csv -b cali.csv

All modules needed: 
- os
- sys
- getopt
- csv
- termios
- signal
- matplotlib.pyplot 
- numpy 
- pandas 
- re
